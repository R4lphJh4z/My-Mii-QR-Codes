# How to Scan Mii QR Codes!
To scan Mii QR Code heres the Step By Step guide:

**1. Access the Scanner:** Open Tomodachi Life and go to the Town Hall, then select the QR code option.

**2. Scan the Code:** Position the 3DS camera to frame the QR code on the top screen.

**3. Import the Mii:** The game will automatically detect the code and import the Mii with their default outfit and interior. 

**Alternative:** Mii Maker Method 

Open the **Mii Maker** application on the 3DS HOME Menu.

Select "**QR Code/Image Options**".

Tap "**Scan QR Code**".

Align the code in the frame. Note: This method imports the Mii's appearance and nickname only, not their clothes or interior. 

Important Notes

**Region Locking:** Clothing items only carry over if the QR code and your game are from the same country.

**Citra Emulator:** On Citra, when prompted to scan, you can select an image file of the QR code from your computer.

**Limitations:** Only one piece of clothing, one interior, and a catchphrase are stored per QR code.
